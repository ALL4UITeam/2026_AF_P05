자주하는 질문

로그인 후 우측 상단 사용자 메뉴의 내 정보 관리에서 비밀번호를 변경할 수 있습니다.

  • 우측 상단 사용자 메뉴 클릭
  • 내 정보 관리 진입
  • 비밀번호 변경 후 저장

레이어 ON/OFF 상태, 축척 범위, 권한 여부를 순서대로 확인해 주세요.

  • 레이어 패널에서 선택 항목 활성화 확인
  • 현재 지도 축척에서 해당 레이어 표시 가능한지 확인
  • 권한 미보유 시 권한 요청 후 재확인

동일 현상이 지속되면 오류 화면 캡처와 함께 문의해 주세요.

  • 권한 요청 메뉴에서 다운로드 권한 신청
  • 신청 사유 및 사용 목적 입력
  • 승인 완료 후 자료 다운로드 가능
